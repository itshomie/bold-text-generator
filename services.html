Simple API Integration

Our JavaScript API makes it easy to add bold text generation to your website or application. Here's a quick example: 

// Import the Bold Text Generator library
import { BoldTextGenerator } from 'bold-text-generator';

// Initialize the generator
const generator = new BoldTextGenerator();

// Convert text to bold
const boldText = generator.convert('Hello World', 'mathBold');

// Get all available styles
const allStyles = generator.getAllStyles('Hello World');

// Batch process multiple strings
const batchResults = generator.batchConvert([
    'First line',
    'Second line',
    'Third line'
], 'mathBold');

// Custom style combination
const customStyle = generator.combine('Text', ['bold', 'italic']);
